# YourCurlyCare

##Requisitos previos:
Para ejecutar el proyecto solo necesitas tener instalado:
* Docker Desktop
* Git

##Pasos para el despliegue:
1. **Clonar el repositorio** --> 	
```bash
 git clone https://github.com/Azaharapp/YourCurlyCare.git
```
2. **Acceder al directorio de trabajo** --> 
```bash
cd YourCurlyCare
```
2. **Ejecutar con Docker Compose** --> 
```bash
docker-compose up --build
```

Para ver el contenido del proyecto, escribe en el navegador "http://localhost:4200"
