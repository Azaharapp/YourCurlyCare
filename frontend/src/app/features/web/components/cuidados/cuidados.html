El último lavado

Antes de empezar a cuidar el rizo, hay que resetearlo. La mayoría de productos contienen siliconas no solubles en agua. Estas siliconas recubren la hebra capilar para que brille y parezca sana, pero en realidad están asfixiando el pelo e impidiendo la hidratación.

El primer paso para iniciar en el método curly es “El último lavado”, se llama así porque es el lavado que se hace para empezar el nuevo proceso para una hidratación y definición de tu cabello. Para ello, necesitas un champú con sulfatos, para eliminar toda la suciedad y plásticos, pero sin siliconas, para no volver a depositar productos nocivos sobre el pelo.

Este lavado es la fase de inicio del método, que además puedes realizar una vez cada mes o después usar un producto prohibido para una profunda limpieza.

La limpieza habitual

Una vez hecho el último lavado, los sulfatos fuertes quedan prohibidos.

Por lo general, el pelo rizado suele ser seco. Si usamos detergentes fuertes a diario, resecamos más el rizo y provocamos frizz.

La técnica adecuada para aplicar el producto es frotando con las yemas de los dedos en el cuero cabelludo. El resto del pelo se limpia con la espuma que cae al aclarar. Frotar las puntas crea enredos innecesarios.

Hidratación y desenredado

Una vez enjuagado el champú, coges una cantidad generosa de acondicionador o mascarilla (una vez por semana, para hidratar).

Frotas el acondicionador entre tus palmas, coges una sección de cabello, colocas una mano plana por debajo y otra plana por encima. Deslizas las manos juntas desde la raíz hasta las puntas.

Ahora, que ya tenemos el acondicionador, empezamos a desenredar, deshaciendo los nudos de abajo y vas subiendo poco a poco hacia la raíz.

A continuación para dar forma a tu rizo, colocas las palmas de las manos hacia arriba, y recoges las puntas del cabello en dirección a la raíz, apretando los mechones contra la cuero cabelludo. Esto se llama scrunch y debe realizarse repetidamente hasta que el cabello tome su forma rizada.

La definición

Justo después de dejar reposar el acondicionador unos minutos, lo enjuagamos por completo. Es en este momento, con el pelo aún empapado, cuando comienza la definición. Hacerlo con el cabello tan mojado es el gran secreto para evitar el encrespamiento desde el minuto cero.

Primero, solemos aplicar un Leave-in (un acondicionador sin aclarado) que actuará como nuestra base hidratante y protectora. A continuación, llega el turno de la fijación, para la cual utilizaremos gel o espuma, dependiendo del peso que soporte nuestro tipo de rizo.

El secado

Una vez terminada la definición, necesitamos retirar el exceso de agua y producto. Para ello, usamos una toalla de microfibra o una camiseta vieja de algodón puro. Hacemos lo que se conoce como micro-plopping: apretamos suavemente el pelo hacia la cabeza con la toalla para absorber el agua sobrante, siempre a toques y sin frotar jamás.

Con el exceso de humedad retirado, podemos dejar que el pelo se seque al aire libre o, si buscamos más volumen y encogimiento, usar el difusor a una temperatura media o baja, y con velocidad suave.
